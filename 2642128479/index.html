Go 语言语法元素

发表于 2024-01-14 更新于 2024-07-23 分类于 Go ， 3.核心概念 阅读次数： 本文字数： 2.9k 阅读时长 ≈ 10 分钟

基本元素

如果将构成源程序的最小单位被称为基本元素，编译过程的第一步是将源码拆解成这些独立的基本元素，这一过程被称为词法分析。在 Go 语言中，基本元素主要分为四类：字面值（literal）、标识符（identifier）、分隔符（delimiter）和操作符（operator），这些元素共同构成了各种表达式和语句。

字面量

字面量（literal）用来表示固定值，它可以表示 Struct（结构体）、Array（数组）、Slice（切片）和 Map（映射）的值。Go 的字面量可以出现在两个地方：一是用于常量和变量的初始化，二是用于表达式里或作为函数调用的实参。

字面量只能表达基本类型的值，按照 Go 语言的基础数据类型，字面量可以分为以下几类：

  • 整型字面量（Integer Literals）：整型数值。例如：123、015321、0x1A3F。
  • 浮点型字面量（Floating-point Literals）：浮点数值。例如：0.1、01.10、1.e+4、1E4、.12E+6。
  • 复数型字面量（Imaginary Literals）：复数类型的常量值。例如：011i、4.1e-18i。
  • 字符型字面量（Rune Literals）：字符常量有多种表现形式，使用单引号「'」括起来。例如：'a'、'杉'、'\t'、'\x5f'、'\u12e3'。
  • 字符串字面量（String Literals）：使用双引号「"」括起来的字符序列，或用反引号「`」括起来的原生字符串字面量。例如："Hello World"、"你好，世界"、"\"\n"。

标识符

在 Go 语言中，标识符用于给变量、函数或自定义类型等对象命名。标识符主要分为两大类：

  • 预定义标识符：这些是由语言的设计者预先定义的，包括语言内置的关键字和特殊标识符。它们涵盖了基础数据类型（如 bool、int、string 等）、内置函数（如 len、print、copy 等）以及特殊符号（如 iota、nil 和占位符 _）。这类标识符是保留字段，不要用作定义常量、变量或任何其他自定义名称。
  • 用户定义标识符：由用户创建的标识符，可以用来命名变量、常量、函数等。在同一个代码块中，标识符必须是唯一的。若要使用不同代码包中的标识符，必须通过包名来访问，如使用 fmt.Println 来调用 fmt 包中的 Println 函数。

关于用户定义标识符的使用有以下规则：

  • 标识符必须以字母（A-Z 和 a-z）或下划线「_」开始，之后可以跟随任意数量的字母、数字（0-9）和下划线，不支持其他特殊字符。
  • 标识符在 Go 语言中是大小写敏感的。
  • 尽管在不同的包中可以使用相同的标识符，但在同一个包内部，每个标识符必须保持唯一。

关键字

关键字指语言设计者保留的有特定语法含义的标识符，主要用于控制程序结构。Go 语言的关键字有 25 个，从发布最初到最新版（1.21）都没有变过，以此保证稳定性和向后兼容性。根据功能分类如下：

  • 引导程序整体结构 8 个：

    关键字 用途
    package 定义包名
    import 导入包名
    const 常量声明
    var 变量声明
    func 定义函数
    defer 延迟执行
    go 并发语法糖
    return 函数返回

  • 声明复合数据结构 4 个：

    关键字 用途
    struct 定义结构类型
    interface 定义接口类型
    map 声明或创建映射类型
    chan 声明或创建通道类型

  • 控制程序结构 13 个：

    关键字 用途
    if、else 条件语句
    for、range、break、continue 循环语句
    switch、select、type、case、default、fallthrough 选择语句
    goto 跳转语句

数据类型

丰富的内置类型支持是高级语言的基本特性，基本类型也是构建自定义类型的基础。Go 语言内置了 20 个预声明数据类型标识符：

  • 数值 16 个：

    数据类型 说明
    byte、int、int8、int16、int32、int64、unit、uint8、uint16、uint32、uint64、uintptr 整形
    float32、float64 浮点型
    complex64、complex128 复数型

  • 字符和字符串型 2 个：string、rune

  • 接口型 1 个：error

  • 布尔型 1 个：bool

Go 是强类型静态编译型语言，在定义新类型或函数时，必须显式地带上数据类型标识符。

内置函数

内置函数具有全局可见性，不需用使用 import 引入。内置函数有 15 个：

内置函数 用途
close 用于关闭通道
len 用于计算某个类型的长度或数量
cap 用于计算切片、数组和通道的最大容量
make、new 用于初始化分配内存。new用于引用以外类型，make用于内置引用类型
append、copy 用于连接和复制切片
delete 用于从切片或映射中删除元素
panic、recover 用于错误处理机制
print、println 用于打印的底层函数
complex、real、imag 用于创建和操作复数

常量值

下面这些常量值表达特殊的含义。共有 4 个：

常量值 用途
true、false 表示布尔类型的真和假
iota 用在连续的枚举类型的声明中
nil 指针或引用型变量的默认值

空白标识符

空白标识符只有 1 个，为下划线「_」，通常用来声明一个匿名变量。在赋值表达式中，空白标识符用作占位，常用于忽略函数某个（多个）返回值。

操作符和分隔符

操作符用于执行算术或逻辑操作，包括运算符、显式的分隔符和其他语法辅助符号。

分隔符用于分隔其他元素。例如 fmt.Println("hello") 中点号、括号和双引号都是分隔符。程序中使用的分隔符有：小括号、中括号、大括号、小数点、逗号、分号、冒号和省略号。

Go 语言使用的分隔符可以分为两大类：

  • 操作符作为分隔符：在 Go 语言中，操作符不仅是基本元素，也充当了分隔符的角色。例如，在表达式 sum := a + b 中，:= 和 + 同时扮演了操作符和分隔符的双重角色，将语句分割成五个独立的基本元素。
  • 非语义分隔符：包括空格、制表符（Tab）、换行符和回车符等。这些分隔符没有具体的语义含义，其主要作用是隔开其他基本元素。多个连续的空格或制表符会被视为单个分隔符。

操作符和分隔符一共有 47 个：

  • 算术运算符 5 个，都是二元运算符：

    符号 说明
    + 相加（求和），二元算术运算符。操作数可以是整数、浮点数和字符串
    - 相减（求差），二元算术运算符。操作数可以是整数和浮点数字
    * 相乘（求积），二元算术运算符。操作数可以是整数和浮点数字。同时也是一元地址运算符，表示取值操作
    / 相除（求商），二元算术运算符。分母不能为 0，操作数为整数既是取整运算
    % 取模（求余），二元算术运算符。操作数为整数，结果为整数

  • 位运算符 6 个，操作数为整数或字节型：

    符号 说明
    ` `
    & 按位与操作，二元算术运算符。同时也是一元地址运算符，表示取址操作
    ^ 按位异或操作，二元算术运算符。同时也是一元运算符，表示按位取补码操作
    << 按位左移操作，二元算术运算符
    >> 按位右移操作，二元算术运算符
    &^ 按位清除操作，二元算术运算符

  • 比较运算符 6 个，用于关系运算：

    符号 说明
    == 等于判断，二元逻辑运算符
    != 不等判断，二元逻辑运算符
    < 小于判断，二元逻辑运算符
    <= 小于等于判断，二元逻辑运算符
    > 大于判断，二元逻辑运算符
    >= 大于等于判断，二元逻辑运算符

  • 括号 6 个：

    符号 说明
    (、) 小括号
    [、] 中括号
    {、} 大括号。左大括号「{」必须出现在与控制语句或函数声明在同一行的末尾

  • 逻辑运算符 3 个，操作元素都是布尔类型：

    符号 说明
    `
    && 与，二元逻辑运算符
    ! 非，一元逻辑运算符，反转操作数逻辑状态

  • 赋值和赋值复核运算符 13 个：

    符号 说明
    := 短变量声明。用于声明并初始化局部变量。例如，x := 1 声明了变量 x 并赋予它值为 1。
    = 简单赋值运算符。用于给变量赋值，就是把等号右边表达式的值赋予等号左边的变量。例如，x = 1 将变量 x 的值设置为 1。
    += 加法赋值运算符。将右侧操作数加到左侧操作数上，然后将结果赋值给左侧操作数。例如，x += 2 相当于 x = x + 2。
    -= 减法赋值运算符。从左侧操作数减去右侧操作数，然后将结果赋值给左侧操作数。
    *= 乘法赋值运算符。将左侧操作数与右侧操作数相乘，然后将结果赋值给左侧操作数。
    /= 除法赋值运算符。将左侧操作数除以右侧操作数，然后将结果赋值给左侧操作数。注意，除以零会导致运行时错误。
    %= 取模赋值运算符。将左侧操作数对右侧操作数取模，然后将结果赋值给左侧操作数。
    &= 按位与赋值运算符。对左右两侧的操作数进行按位与操作，然后将结果赋值给左侧操作数。
    ` =`
    ^= 按位异或赋值运算符。对左右两侧的操作数进行按位异或操作，然后将结果赋值给左侧操作数。
    &^= 按位清除赋值运算符。将左侧操作数的位中，对应右侧操作数位为 1 的部分设置为 0，然后将结果赋值给左侧操作数。
    >>= 右移赋值运算符。将左侧操作数向右移动右侧操作数指定的位数，然后将结果赋值给左侧操作数。
    <<= 左移赋值运算符。将左侧操作数向左移动右侧操作数指定的位数，然后将结果赋值给左侧操作数。

  • 自增自减操作符 2 个。属于语句而非表达式，没有运算符优先级之说：

    符号 说明
    ++ a++ 等同于 a = a + 1。不支持前置增量操作 ++a
    -- a-- 等同于 a = a - 1。不支持前置减量操作 --a

  • 其他运算符 6 个：

    符号 说明
    : 常见于 := 给变量赋值
    , 常用于分隔参数
    ; 用于表示语句结束
    . 常用于分隔包和函数
    ... 用于表示不定参数
    <- 用于通道数据指向，接受或发送数据

运算符优先级

一元运算符优先级比二元运算符高。二元运算符优先级如下：

运算符 优先级
* / % << >> & &^ 最高
+ - | ^ 较高
== != < <= > >= 正常
&& 较低
`

其他运算符规则：

  • 如果一个表达式中出现相同优先级的运算符，那么按照从左到右顺序依次操作；
  • 所有运算符都受括号影响，括号内先操作。在复杂表达式中，为了避免优先级引起的混淆，建议使用括号明确操作顺序；
  • Go 语言不支持运算符重载。
