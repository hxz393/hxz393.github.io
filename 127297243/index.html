Go 语言扩展数据结构

发表于 2024-07-17 更新于 2024-07-23 分类于 Go ， 4.基本应用 阅读次数： 本文字数： 1.9k 阅读时长 ≈ 6 分钟

container 包

在 container 包中实现了三个复杂的数据结构：堆（heap）、列表（list）和环（ring），在处理特定类型的数据和算法时可以直接拿来使用。

堆

堆是一种特殊的树形数据结构，通常用来实现优先队列。Go 的 container/heap 包提供了接口和实现，可以轻松地在任何满足接口的切片上实现堆操作。

最小堆（Min Heap）和最大堆（Max Heap）是堆的两种主要类型，主要在优先级队列、排序算法等场景中使用。核心特性是树的根节点（顶部节点）总是最小值或最大值，非常适合快速访问和删除最小或最大元素的操作。

堆提供的功能包括：

  • 插入（Push）：向堆中添加新元素。
  • 删除最小（或最大）元素（Pop）：移除堆顶元素，即最小（或最大）元素。
  • 构建堆（Init）：将一个不满足堆性质的数组转换成一个堆结构。
  • 删除任意元素（Remove）：移除堆中的任意元素。
  • 修复堆（Fix）：当堆中某个元素值被修改后，重新调整以维持堆性质。

Go 的 container/heap 包提供了一个 Interface 接口类型，包含必需的方法：

  • Len()：返回堆中的元素数量。
  • Less(i, j int) bool：判断元素 i 是否应该排在元素 j 之前。
  • Swap(i, j int)：交换元素 i 和 j 的位置。
  • Push(x interface{})：向堆中添加元素 x。
  • Pop() interface{}：从堆中移除并返回最后一个元素。

下面是一个简单的最小堆：

package main

import (
	"container/heap"
	"fmt"
)

// IntHeap 是一个由整数组成的最小堆
type IntHeap []int

// 定义来自 sort 接口要求的方法
func (h *IntHeap) Len() int           { return len(*h) }
func (h *IntHeap) Less(i, j int) bool { return (*h)[i] < (*h)[j] }
func (h *IntHeap) Swap(i, j int)      { (*h)[i], (*h)[j] = (*h)[j], (*h)[i] }

// Push 方法向切片末尾插入元素
func (h *IntHeap) Push(x interface{}) {
	*h = append(*h, x.(int))
}

// Pop 方法重新生成一个切片，去除最后一个元素，返回被删除的元素
func (h *IntHeap) Pop() interface{} {
	old := *h
	n := len(old)
	x := old[n-1]
	*h = old[0 : n-1]
	return x
}

func main() {
	// 创建一个新的最小堆并添加元素
	h := &IntHeap{2, 3, 1, 5, 4}
	// 初始化堆，保证 h[0] 是最小值
	heap.Init(h)
	heap.Push(h, 6)
	fmt.Printf("minimum: %d\n", (*h)[0])

	// 逐个弹出元素
	for h.Len() > 0 {
		fmt.Printf("%d ", heap.Pop(h))
	}
}

列表

列表是一种非连续储存的容器，由多个节点组成，节点通过一些变量记录彼此之间的关系。列表有多种实现方法，如单链表、双链表等。由 container/list 包提供的列表是双向链表结构，双向链表能高效地操作插入和移除，不需要进行大量的元素移动。

双向链表由一系列节点组成，每个节点都包含数据和两个链接：一个指向前一个节点，一个指向后一个节点。在 Go 的 list 包中，链表由 List 结构体表示，每个节点是一个 Element 结构体实例，具有以下属性：

  • Value：存储节点的实际数据，类型为 interface{}。
  • Next 和 Prev：分别指向链表中的下一个和前一个节点。

list 包中的主要类型和方法包括：

  • List 类型：表示链表本身。
  • PushFront 和 PushBack：在链表的前端或后端插入元素。
  • Remove：移除链表中的元素。
  • Front 和 Back：分别获取链表的第一个和最后一个元素。

初始化列表

初始化列表有两种方法 变量名 := list.New() 和 var 变量名 list.List，两种方法初始化效果相同。列表的元素可以是任意类型：

package main

import (
	"container/list"
	"fmt"
)

func main() {
	myList := list.New()
	var yourList list.List

	fmt.Println(myList)   // 输出：&{{0xc0420fe1e0 0xc0420fe1e0 <nil> <nil>} 0}
	fmt.Println(yourList) // 输出：{{<nil> <nil> <nil> <nil>} 0}
}

插入元素

双链表支持从队列前方或后方插入元素，返回 *list.Element 结构：

package main

import (
	"container/list"
	"fmt"
)

func main() {
	myList := list.New()
    // 在链表的前端插入一个新元素
	first := myList.PushFront("1")
    // 在链表的末尾插入一个新元素
	last := myList.PushBack(-1)
    // 在链表中的特定位置插入新元素
	myList.InsertAfter(0, first)

	fmt.Printf("%#v\n%#v", first, last)
}

删除元素

删除元素需要用到插入元素时的返回值：

package main

import (
	"container/list"
	"fmt"
)

func main() {
	myList := list.New()
	first := myList.PushFront("1")
	myList.Remove(first)
	fmt.Printf("%#v", myList)
}

遍历列表

遍历双链表需要配合 Front 函数获取头元素，遍历时只要元素不为空就可以继续进行。每次遍历调用元素的 Next 方法继续：

package main

import (
	"container/list"
	"fmt"
)

func main() {
	myList := list.New()
	myList.PushFront("1")
	myList.PushFront("0")

	for i := myList.Front(); i != nil; i = i.Next() {
		fmt.Println(i.Value)
	}
}

环

环是一种循环链表结构，由 container/ring 包提供。每个元素都指向下一个，最后一个元素指向第一个，形成一个闭环。环结构可以无限循环遍历，适用于需要循环访问元素的场景，如轮转调度等。

环形链表的主要特性包括：

  • 无明显的起始或结束节点：链表形成一个闭环，可以从任何一个节点开始遍历整个链表。
  • 固定大小：环的大小是预设的，但可以动态增加或减少节点。
  • 灵活的节点操作：可以容易地添加、删除或移动节点。

ring 包提供的功能包括：

  • 创建：通过 ring.New(n int) 创建一个有 n 个节点的环。
  • 遍历：使用 Do 方法对环中的每个元素执行一个给定的函数。
  • 移动：Move(n int) 方法可以将环的当前位置向前或向后移动 n 步。
  • 链接：Link(r *Ring) 方法可以将另一个环 r 链接到当前环之后。
  • 拆分：Unlink(n int) 方法从当前环的下一个位置开始，移除 n 个元素，并返回一个新的环。

下面是使用 Go 语言中的 container/ring 包进行环形链表操作的五个示例，涵盖创建、遍历、移动、链接和拆分：

创建环

使用 ring.New 创建指定节点数的环：

package main

import (
	"container/ring"
	"fmt"
)

func main() {
	// 创建一个包含 5 个元素的环
	r := ring.New(5)

	// 初始化环的元素值，没赋值的为 nil
	for i := 1; i <= 4; i++ {
		r.Value = i
		r = r.Next()
	}
	// 打印环长度，和实际赋值数量无关
	fmt.Println(r.Len()) // 输出：5
}

遍历环

使用 Do 方法对环中的每个元素执行一个函数：

package main

import (
	"container/ring"
	"fmt"
)

func main() {
	r := ring.New(2)
	// 分别赋值
	r.Value = 1
	r.Next().Value = 20
    
	// 计算环中所有元素的总和
	sum := 0
	r.Do(func(p interface{}) {
		sum += p.(int)
	})
	fmt.Println("Sum of values:", sum) // 输出：21
}

移动环

使用 Move 方法来移动环的当前位置：

package main

import (
	"container/ring"
	"fmt"
)

func main() {
	r := ring.New(3)
	for i := 1; i <= 3; i++ {
		r.Value = i
		r = r.Next()
	}
	fmt.Println("Current value :", r.Value) // 输出：1

	// 向前移动 2 步
	r = r.Move(2)
	fmt.Println("Current value after moving 2 steps forward:", r.Value) // 输出：3
	// 向后移动 1 步
	r = r.Move(-1)
	fmt.Println("Current value after moving 1 steps backward:", r.Value) // 输出：2
}

链接环

使用 Link 方法将一个环链接到另一个环：

package main

import (
	"container/ring"
	"fmt"
)

func main() {
	r1 := ring.New(2)
	r2 := ring.New(2)

	r1.Value = "A"
	r1.Next().Value = "B"
	r2.Value = "C"
	r2.Next().Value = "D"

	// 链接两个环
	r1.Link(r2)
	fmt.Println("Combined ring:")
	r1.Do(func(p interface{}) {
		fmt.Println(p) // 分别输出：A C D B
	})
}

拆分环

使用 Unlink 方法从当前环移除指定元素，并返回一个包含移除元素的新环：

package main

import (
	"container/ring"
	"fmt"
)

func main() {
	r := ring.New(5)
	for i := 1; i <= 5; i++ {
		r.Value = i
		r = r.Next()
	}

	// 从当前位置开始移除 3 个元素
	newRing := r.Unlink(3)

	// 定义打印函数，并分别作用于新旧环
	f := func(p interface{}) { fmt.Println(p) }

	fmt.Println("Original ring after unlink:")
	r.Do(f) // 分别输出：1 5

	fmt.Println("New ring:")
	newRing.Do(f) // 分别输出：2 3 4
}
