Go 语言中使用正则表达式

发表于 2024-06-20 更新于 2024-07-23 分类于 Go ， 4.基本应用 阅读次数： 本文字数： 836 阅读时长 ≈ 3 分钟

正则表达式引擎

Go 语言中的正则表达式功能由 regexp 包提供，regexp 包使用 RE2 正则表达式引擎，大致与 Perl 和 Python 语法相同。

Go 语言里的正则引擎不支持反向引用，不能在正则表达式中使用 \1, \2 等来引用捕获组。

编译正则表达式

使用 regexp.Compile 函数编译一个正则表达式，返回一个正则表达式对象和错误。如果要确保正则表达式编译必须成功，可以使用 regexp.MustCompile 函数来编译，失败时会引发 panic：

package main

import (
	"fmt"
	"regexp"
)

func main() {
	// 使用 regexp.Compile 编译正则表达式
	re, err := regexp.Compile(`\d+`)
	if err != nil {
		fmt.Println("预编译正则表达式时出错:", err)
		return
	}

	// 使用 regexp.MustCompile 编译正则表达式，确保表达式正确
	re = regexp.MustCompile(`\d+`)

	fmt.Println("编译好的正则表达式:", re)
}

检查匹配结果

最简单的是用 MatchString 方法检查一个字符串是否匹配一个正则表达式，返回一个布尔值：

package main

import (
	"fmt"
	"regexp"
)

func main() {
	re := regexp.MustCompile(`^hello`)

	// 使用 MatchString 方法检查参数中的字符串是否匹配 re 正则表达式
	match := re.MatchString("hello, world")
	fmt.Println(match) // 输出：true
	fmt.Println(re.MatchString("world")) // 输出：false
}

忽略大小写

想要让正则表达式匹配时忽略字母大小写，可在正则表达式最前面使用 (?i) 来忽略大小写：

package main

import (
	"fmt"
	"regexp"
)

func main() {
	re := regexp.MustCompile(`(?i)^hellO`)      // 全部忽略大小写
	fmt.Println(re.MatchString("Hello, world")) // 输出：true
}

匹配字符串

在字符串中搜索与正则表达式匹配的子串，可以使用 FindString。方法返回字符串中第一个匹配的子字符串，如果没有匹配则返回空字符串：

package main

import (
	"fmt"
	"regexp"
)

func main() {
	re := regexp.MustCompile(`\d+`)
	result := re.FindString("There are 123 apples and 456 oranges")
	fmt.Println("匹配到第一个结果:", result)                          // 只会返回 123
	fmt.Println("没有匹配结果：", re.FindString("There are apples")) // 没有结果
}

如果要返回所有匹配的子字符串，可以使用 FindAllString。方法返回一个切片，包含所有匹配的非重叠子字符串，如果没有匹配则返回 nil：

package main

import (
	"fmt"
	"regexp"
)

func main() {
	re := regexp.MustCompile(`\d+`)
	results := re.FindAllString("There are 123 apples and 456 oranges", -1)
	fmt.Println("All matches found:", results) // 返回 [123 456]
}

匹配索引

有时需要获取匹配字符串所在的索引位置，可以使用 FindStringIndex 方法：

package main

import (
	"fmt"
	"regexp"
)

func main() {
	re := regexp.MustCompile(`\d+`)
	result := re.FindStringIndex("There are 123 apples and 456 oranges")
	fmt.Println("匹配索引结果:", result) // 返回所有匹配位置：[10 13]
}

替换匹配

使用 ReplaceAllString 方法可以替换字符串中的所有匹配项：

package main

import (
	"fmt"
	"regexp"
)

func main() {
	r := regexp.MustCompile(`\d+`)
	fmt.Println(r.ReplaceAllString("abc 123 def 45", "XYZ")) // 将所有匹配项替换结果：abc XYZ def XYZ
}

Go 虽然不支持反向引用，但可以通过 ReplaceAllStringFunc 方法来操作捕获组：

package main

import (
	"fmt"
	"regexp"
)

func main() {
	r := regexp.MustCompile(`(\d+)`)
	input := "user id: 12, user age: 1"
	replacement := "100"

	// 对匹配到的字符串应用函数
	result := r.ReplaceAllStringFunc(input, func(match string) string {
		return replacement + match
	})

	fmt.Println(result) // 输出：user id: 10012, user age: 1001
}

如果要单独操作多个匹配组，可以尝试 FindAllStringSubmatchIndex 方法。

分割字符串

根据匹配项分割字符串：

package main

import (
	"fmt"
	"log"
	"regexp"
)

func main() {
	r, err := regexp.Compile(`\d+`)
	if err != nil {
		log.Fatal(err)
	}

	fmt.Println(r.Split("abc 123 def 456 ghi", -1)) // 分割字符串结果：[abc   def   ghi]
}
