Go 语言创建 Socket 服务

发表于 2024-06-27 更新于 2024-07-23 分类于 Go ， 5.网络编程 阅读次数： 本文字数： 1.2k 阅读时长 ≈ 4 分钟

Socket 编程

传统的 Socket 编程通常涉及以下步骤：

  • 服务器端：

    1. 创建 Socket；
    2. 绑定 Socket 到本地地址和端口；
    3. 监听连接；
    4. 接受连接；
    5. 读取或写入数据；
    6. 关闭 Socket；

  • 客户端：

    1. 创建 Socket；
    2. （可选）绑定本地地址；
    3. 发起连接到服务器；
    4. 读取或写入数据；
    5. 关闭 Socket；

建立连接

Go 语言对传统 Socket 网络编程过程进行了封装，只需要调用 net.Dial 函数即可完成协议选择和建立连接：

package main

import (
	"bufio"
	"fmt"
	"net"
)

func main() {
	// 指定服务器的主机名和端口号
	server := "127.0.0.1:808"

	// 使用 net.Dial 创建 TCP 连接
	conn, _ := net.Dial("tcp", server)
	defer conn.Close() // 确保在main函数结束前关闭连接
	fmt.Println("Connected to", server)

	// 发送数据到服务器
	message := "Hello, Server!"
	fmt.Fprintf(conn, message+"\n")
	
	// 读取服务器的响应
	response, _ := bufio.NewReader(conn).ReadString('\n')
	fmt.Println("Server response:", response)
}

基本方法

无论是服务器还是客户端，数据的发送和接收都通过 net.Conn 接口进行。接口有 3 个基本方法：

  • Read(b []byte) (n int, err error)：从连接中读取数据，并写入到字节数组 b 中。返回实际读取到的字节数和错误。
  • Write(b []byte) (n int, err error)：向连接写入数据 b。
  • Close() error：关闭连接。

TCP Socket

Go 语言中对 TCP socket 的支持是通过标准库中的 net 包实现。在服务器和客户端之间建立连接后，连接一直可用，直到任一端关闭连接。

TCP 服务器

创建一个 TCP 服务器需要以下步骤：

  • 监听端口：使用 net.Listen("tcp", address) 函数开始在一个指定的地址和端口上监听。
  • 接受连接：调用 listener.Accept 等待并接受来自客户端的连接。这个调用是阻塞的，直到有一个新的连接建立。
  • 处理连接：对于每个接受的连接使用 goroutine 来读取客户端数据，进行处理，然后发送响应。

下面展示一个简单功能的 TCP 服务器：

package main

import (
	"fmt"
	"io"
	"net"
	"time"
)

func main() {
	// 监听本地 8080 端口，返回监听器
	listener, _ := net.Listen("tcp", "localhost:8080")
	defer listener.Close()
	fmt.Println("Server is listening on localhost:8080")

	// 循环处理来自客户端的连接
	for {
		// 调用成功后，返回一个可读写的 conn 对象
		conn, err := listener.Accept()
		if err != nil {
			fmt.Println("Error accepting:", err)
			continue
		}
		// 并发处理每个客户端请求
		go handleConnection(conn)
	}
}

// 模拟处理客户端发送数据
func handleConnection(conn net.Conn) {
	defer conn.Close()
	buf := make([]byte, 512)

	for {
		// 设置统一超时时间
		conn.SetDeadline(time.Now().Add(5 * time.Second))
		n, err := conn.Read(buf)
		if err != nil {
			// EOF 代表内容的结束
			if err != io.EOF {
				fmt.Println("Error reading:", err)
			}
			fmt.Println()
			break
		}
		fmt.Printf("Received from %s: %s", conn.RemoteAddr(), string(buf[:n]))
		conn.Write([]byte("Received your message\n"))
	}
}

TCP 客户端

客户端的创建相对简单：

  • 连接服务器：使用 net.Dial("tcp", serverAddress) 连接到服务器，返回 Conn 类型对象。
  • 发送和接收数据：通过 Conn 连接对象来发送和接收数据。

创建一个客户端，向刚才的服务端发起连接：

package main

import (
	"fmt"
	"net"
	"time"
)

func main() {
	// 和服务端建立连接，成功时返回 conn 对象
	conn, _ := net.Dial("tcp", "localhost:8080")
	// 加入超时参数，并且地址传入域名
	conn, _ = net.DialTimeout("tcp", "www.baidu.com:8081", time.Second*5)
	defer conn.Close()

	// 发送消息通过 Write 方法
	message := "Hello, Server!"
	conn.Write([]byte(message))

	// 读取回应通过 Read 方法
	reply := make([]byte, 512)
	n, _ := conn.Read(reply)
	fmt.Println("Server says:", string(reply[:n]))
}

UDP Socket

使用 UDP 的 Socket 与 TCP 的主要区别在于 UDP 是一种无连接的协议，不需要维持一个保持的连接。因此特别提供了 ReadFromUDP 和 WriteToUDP 方法，直接使用服务端地址进行数据发送和接受，也不管上次通信是否完成或异常，依然会接受下一次连接请求。

UDP 服务器

这里使用 net.ListenUDP 来监听启动 UDP 服务器：

package main

import (
	"fmt"
	"net"
)

func main() {
	// 转为 UDPAddr 类型
	addr := "localhost:8080"
	udpAddr, _ := net.ResolveUDPAddr("udp", addr)

	// 监听端口，返回 UDPConn 对象
	conn, _ := net.ListenUDP("udp", udpAddr)
	defer conn.Close()
	fmt.Println("UDP server listening on", addr)
	
	// 循环读取数据并回显
	buffer := make([]byte, 1024)
	for {
		n, clientAddr, err := conn.ReadFromUDP(buffer)
		if err != nil {
			fmt.Println("Error reading from UDP:", err)
			continue
		}
		fmt.Printf("Received '%s' from %s\n", string(buffer[:n]), clientAddr)
		// 返回给客户端
		conn.WriteToUDP(buffer[:n], clientAddr)
	}
}

UDP 客户端

由于 UDP 不保证数据的可靠性，客户端通常需要设置连接超时和数据重传逻辑：

package main

import (
	"fmt"
	"net"
	"time"
)

func main() {
	// 建立连接，返回可操作的 UDPConn 对象
	serverAddr := "localhost:8080"
	udpAddr, _ := net.ResolveUDPAddr("udp", serverAddr)
	conn, _ := net.DialUDP("udp", nil, udpAddr)
	defer conn.Close()

	// 发送消息
	message := "Hello, UDP Server!"
	conn.Write([]byte(message))

	// 读取返回
	buffer := make([]byte, 1024)
	conn.SetReadDeadline(time.Now().Add(5 * time.Second))
	n, _ := conn.Read(buffer)
	fmt.Printf("Received from server: %s\n", string(buffer[:n]))
}
