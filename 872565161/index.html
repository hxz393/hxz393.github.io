Go 语言类型：数组

发表于 2024-04-11 更新于 2024-07-23 分类于 Go ， 2.类型系统 阅读次数： 本文字数： 1.7k 阅读时长 ≈ 6 分钟

数组定义

数组（Array）是具有相同类型的一组定长且有序的变量集合，存放于内存中一块连续区域内。组成数组的各个变量称为数组的元素（element）。每个数组元素都有一个编号，称为数组下标或索引（Array-index）。

数组包含元素的个数为数组长度，是数组类型的一部分，而类型必须在编译时确定，因此不能通过变量去修改数组长度。不同长度的数组被认为是不同类型，所以它们之间也不能互相转换。

数组属于值类型，一旦声明，数组的数据类型和长度都不可修改。由于切片概念的引入，所以在 Go 语言中不容易见到数组，但数组是切片和映射的基础数据结构。

声明数组

声明数组需要指定元素的数据类型和数组的长度，常用方法如下：

  • var 数组名 [数组长度]数据类型 = [数组长度]数据类型{值1, 值2, ...}：完整声明格式，声明数组后同时赋值。
  • 数组名 := [数组长度]数据类型{值1, 值2, ...}：简短声明格式。
  • 数组名 := new([数组长度]数据类型)：使用 new 函数创建数组，并返回指向这块内存的指针。当用 new 创建数组时，实际上是在创建一个指向数组的指针，返回的是指向该数组的指针，而不是数组本身。

其中数组长度和元素值可以有不同定义方法：

  • var 数组名 [数组长度]数据类型：只声明数组变量，数组元素的值为初始化零值，之后可以赋值修改。

  • 数组名 := [...]数据类型{值1, 值2, ...}：如果有定义数组元素的值，可使用 ... 替代数组长度，由编译器自动推断长度。

  • 数组名 := [...]数据类型{索引1: 索引值1, 索引2: 索引值2, ...}：可用数组下标来定义值，没有定义的数组下标值为类型零值，数组的长度由提供的最高索引决定（或手动指定数组长度）。

  • 数组名 := [...]数据类型{值1, 索引1: 索引值1, 索引2: 索引值2, 值2, 值3, ...}：混合定义用法，其中 值1 索引为 0，值2 的索引为紧跟着之前索引顺序初始化，等于 索引2 加一，后面的 值3 继续按顺序初始化，索引为 值2 的索引加一。

下面是几种声明方式完整示例：

package main

import "fmt"

func main() {
	// 完整声明格式，同时初始化数组
	var fullArray [5]int = [5]int{1, 2, 3, 4, 5}
	fmt.Println("完整声明数组并赋值：", fullArray)

	// 只声明数组，元素值为数据类型的零值
	var emptyArray [5]int
	fmt.Println("声明数组，元素为零值：", emptyArray)

	// 简短声明格式，同时初始化数组
	shortArray := [5]int{6, 7, 8, 9, 10}
	fmt.Println("简短声明数组并赋值：", shortArray)

	// 使用索引初始化数组元素
	indexedArray := [5]int{0: 16, 2: 17, 4: 18}
	fmt.Println("使用索引初始化数组元素：", indexedArray)

	// 省略数组长度，由编译器推断
	inferredArray := [...]int{11, 12, 13, 14, 15}
	fmt.Println("省略数组长度，由编译器推断：", inferredArray)

	// 混合赋值模式
	mixedArray := [...]int{1, 5: 5, 2: 2, 6, 7}
	fmt.Println("混合赋值结果：", mixedArray)
}

遍历数组

数组通过数组下标索引位置来读取或修改元素，索引从 0 开始。此外常使用 for 循环来遍历数组：

package main

import (
	"fmt"
)

func main() {
	numbers := [2]int{1, 2}

	// 使用传统的 for 循环通过索引遍历数组
	fmt.Println("使用传统 for 循环遍历数组：")
	for i := 0; i < len(numbers); i++ {
		fmt.Printf("索引：%d, 值：%d\n", i, numbers[i])
	}

	// 使用 range 关键字遍历数组，同时获取索引和元素的值
	fmt.Println("使用 range 遍历数组：")
	for i, v := range numbers {
		fmt.Printf("索引：%d, 值：%d\n", i, v)
	}

	// 使用 range 关键字遍历时，可以只获取索引编号
	fmt.Println("使用 range 遍历索引：")
	for i := range numbers {
		fmt.Printf("索引值：%d\n", numbers[i])
	}
}

修改数组

可以直接通过数组索引，对数组中的元素重新赋值。索引必须位于边界范围之内，且索引只能为正整数，也就是不支持从末尾开始索引：

package main

import "fmt"

func main() {
	// 创建一个包含两个整数的数组
	numbers := [2]int{1, 2}

	// 打印数组的第一个元素的初始值
	fmt.Println("初始第一个元素的值：", numbers[0])

	// 修改数组的第一个元素
	numbers[0] = 10

	// 打印数组的第一个元素修改后的值
	fmt.Println("修改后第一个元素的值：", numbers[0])

	// 尝试访问或修改超出数组边界的元素将导致编译时错误
	//numbers[2] = 11 // 报错：invalid array index 2 (out of bounds for 2-element array)

	// 不支持倒序索引
	//numbers[-1] = 11 // 报错：invalid array index -1 (index must be non-negative)
}

传递数组

下面示例展示在函数声明和函数调用中声明和传递数组：

package main

import "fmt"

func main() {
	// 调用函数，传递数组参数
	resultArray := doubleArrayValues([...]int{1, 2, 3, 4, 5})
	fmt.Println("函数返回的数组：", resultArray)
}

// doubleArrayValues 函数接收一个整型数组，并返回一个新数组，其中每个元素值是原数组值的两倍
func doubleArrayValues(inputArray [5]int) [5]int {
	var result [5]int
	for i, value := range inputArray {
		result[i] = value * 2
	}
	return result
}

由于数组是值类型，因此传递数组时会传递数据副本：

package main

import "fmt"

func main() {
	// 初始化两个整型数组
	initialArray1 := [2]int{1, 2}
	initialArray2 := [2]int{3, 4}

	// 将两个数组嵌套为一个新的二维数组，此操作会创建数组的副本
	nestedArray := [2][2]int{initialArray1, initialArray2}

	// 修改 initialArray2 的第一个元素
	initialArray2[0] = 100

	// 展示 nestedArray 的值，确认修改 initialArray2 不会影响 nestedArray
	fmt.Printf("initialArray1: %v, initialArray2（modified）: %v, nestedArray: %v\n", initialArray1, initialArray2, nestedArray)
}

当要传递的数组很大时，可以考虑传递数组的指针或数组的切片。

多维数组

数组本身只有一个维度，但可以组合多个数组创建多维数组（Multidimensional-array）。数组的维度是指数组的下标个数，多维数组有多个下标。多维数组中每个元素也是类型相同的一维数组，常用于管理具有依赖关系的数据（例如坐标）。下面是一个三维数组示例：

package main

import "fmt"

func main() {
	// 构造一个 2x2x2 的三维数组
	threeDArray := [2][2][2]int{
		{
			{1, 2},
			{3, 4},
		},
		{
			{5, 6},
			{7, 8},
		},
	}

	// 访问并打印坐标为(1, 0, 1)的元素
	fmt.Printf("位于(1, 0, 1)的元素值为：%d\n", threeDArray[1][0][1])

	// 将坐标为(1, 1, 1)的元素修改为 99
	threeDArray[1][1][1] = 99

	// 再次访问并打印被修改后的元素
	fmt.Printf("修改后，整个三维整型数组值为：%v\n", threeDArray)
}

虽然数组 threeDArray 可以存放 2x2x2=8 个整型元素，但它的长度为 2。
