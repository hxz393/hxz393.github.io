使用 Go 语言加密解密

发表于 2024-07-20 更新于 2024-07-23 分类于 Go ， 4.基本应用 阅读次数： 本文字数： 1.5k 阅读时长 ≈ 5 分钟

Base64

Base64 通常用于将二进制数据转换为文本表示，例如在网络中传输图片，先使用 Base64 编码转为文本字符串进行传输，接受方再对字符串进行解码还原为图片。

基本过程如下：

  • 编码过程：Base64 编码将每三个字节的二进制数据转换成四个字节的文本数据。因为每个 Base64 字符存储的是 6 位，所以三个字节可以刚好转换为四个 6 位的 Base64 字符。

  • 解码过程：Base64 解码是将编码的字符串转换回原始的二进制数据。

标准用法

在 Go 语言中使用 encoding/base64 包编码和解码 Base64 数据：

package main

import (
	"encoding/base64"
	"fmt"
)

func main() {
	// 要编码的数据
	data := "hello world"

	// 编码
	sEnc := base64.StdEncoding.EncodeToString([]byte(data))
	fmt.Println("Encoded:", sEnc) // 输出：aGVsbG8gd29ybGQ=

	// 解码
	sDec, _ := base64.StdEncoding.DecodeString(sEnc)
	fmt.Println("Decoded:", string(sDec)) // 输出：aGVsbG8gd29ybGQ=
}

避免 URL 转义

标准 Base64 编码使用的字符集包括字母 A-Z、a-z、数字 0-9 和两个额外符号 + 和 /。在 Base64 URL 安全编码中 + 和 / 字符替换为 - 和 _ 无歧义字符，以避免 URL 转义问题：

package main

import (
	"encoding/base64"
	"fmt"
)

func main() {
	// 这个数据会在标准 Base64 编码结果中产生 + 和 / 字符
	data := []byte{0xfb, 0xff, 0xbf}

	// 使用标准 Base64 编码，原样输出
	stdEncoded := base64.StdEncoding.EncodeToString(data)
	fmt.Println("Standard Base64 Encoded:", stdEncoded) // 输出：+/+/

	// 使用 URL 安全的 Base64 编码，会被自动替换
	urlEncoded := base64.URLEncoding.EncodeToString(data)
	fmt.Println("URL Safe Base64 Encoded:", urlEncoded) // 输出：-_-_
}

随机数

Go 语言中，生成伪随机数主要使用 math/rand 包，如果需要加密安全的随机数，使用 crypto/rand 包。

math/rand

使用 rand 可以生成随机整数和浮点数。~~使用前需要通过 rand.Seed 来设置种子，否则每次程序运行时生成的随机数将不变，种子通常可以使用当前时间来初始化。~~在 Go 1.20 以上版本中，不再需要设置随机数种子：

package main

import (
	"fmt"
	"math/rand"
)

func main() {
	//rand.Seed(time.Now().UnixNano()) // 设置随机数种子

	fmt.Println("Random Int:", rand.Intn(100))     // 生成一个 0 到 100 之间的随机整数
	fmt.Println("Random Float64:", rand.Float64()) // 生成一个 0.0 到 1.0 之间的随机浮点数
}

crypto/rand

对于需要高安全性的应用，如加密、安全令牌生成等，应使用 crypto/rand 包，它提供基于操作系统的安全随机数生成器：

package main

import (
	"crypto/rand"
	"fmt"
)

func main() {
	b := make([]byte, 16)  // 创建一个长度为 16 的字节切片
	_, err := rand.Read(b) // 生成随机字节
	if err != nil {
		fmt.Println("Error:", err)
		return
	}
	fmt.Printf("Crypto Random Bytes: %x\n", b) // 输出随机字节的十六进制表示
}

加密算法

在 Go 语言的 crypto 包中，支持各种常用加密算法：

  • crypto/aes：提供 AES 加密的实现。
  • crypto/cipher：包含用于加密的接口和一些基本的实现，比如块模式的加密。
  • crypto/des：提供 DES 和 Triple DES 加密的实现。
  • crypto/dsa：提供 DSA（数字签名算法）的实现。
  • crypto/ecdsa：实现 ECDSA（椭圆曲线数字签名算法）。
  • crypto/ed25519：用于 Ed25519 签名算法，这也是一种使用椭圆曲线的签名算法。
  • crypto/hmac：实现了基于哈希的消息认证码（HMAC）。
  • crypto/md5、crypto/sha1、crypto/sha256 等：提供不同的哈希算法。
  • crypto/rsa：实现 RSA 加密和签名算法。
  • crypto/x509：用于 X.509 公钥证书和密钥的解析和序列化。

MD5 哈希

MD5 是一种广泛使用的哈希函数，用于生成数据的哈希值：

package main

import (
	"crypto/md5"
	"fmt"
	"io"
)

func main() {
	data := "Hello, world! This is a test of MD5 hash."
	hasher := md5.New()
	io.WriteString(hasher, data)
	hash := hasher.Sum(nil)            // 生成的哈希值是固定的
	fmt.Printf("MD5 Hash: %x\n", hash) // 输出：d9ab810aba33d8db862cb53fc27efcb2
}

MD5 设计为能快速计算，然而在现在高性能计算机上非常容易被暴力破解，并且容易受到碰撞攻击，也就是能让不同的文件具有相同的哈希值。同理 SHA-1 也容易收到碰撞攻击，因此不再推荐使用，只用于向后兼容老系统。

SHA-3 哈希

SHA-3 是 SHA 家族中最新的安全散列算法，提供更强的安全性，更能抵抗碰撞攻击：

package main

import (
	"fmt"
	"golang.org/x/crypto/sha3"
)

func main() {
	data := "hello world"
	hasher := sha3.New256() // 可以选择 SHA-3 的不同位数版本，如 New512, New384, New256
	hasher.Write([]byte(data))
	hash := hasher.Sum(nil)
	fmt.Printf("SHA-3 Hash: %x\n", hash) // 输出：644bcc7e564373040999aac89e7622f3ca71fba1d972fd94a31c3bfbf24e3938
}

所有哈希算法都实现了同样的 hash.Hash 接口，可以用相同的模式来使用这些哈希函数。

AES 加密

AES 是一种对称密钥加密算法，即加密和解密使用同一个密钥，类似的对称加密算法还有 DES。AES 加密需要配合模式使用：

package main

import (
	"bytes"
	"crypto/aes"
	"crypto/cipher"
	"crypto/rand"
	"fmt"
	"io"
)

func main() {
	// 生成一个密钥（AES-256 需要 32 字节大小）
	key := make([]byte, 32)
	_, err := rand.Read(key)
	if err != nil {
		fmt.Println("Key generation failed:", err)
		return
	}

	plaintext := []byte("hello world") // 待加密的数据
	// 对数据进行 PKCS#7 填充
	blockSize := aes.BlockSize
	padding := blockSize - len(plaintext)%blockSize
	paddedText := append(plaintext, bytes.Repeat([]byte{byte(padding)}, padding)...)

	block, err := aes.NewCipher(key)
	if err != nil {
		fmt.Println("Key error:", err)
		return
	}

	// AES 的加密数据块大小固定
	ciphertext := make([]byte, aes.BlockSize+len(paddedText))
	iv := ciphertext[:aes.BlockSize] // 初始向量
	if _, err := io.ReadFull(rand.Reader, iv); err != nil {
		fmt.Println("IV generation failed:", err)
		return
	}

	mode := cipher.NewCBCEncrypter(block, iv)
	mode.CryptBlocks(ciphertext[aes.BlockSize:], paddedText)

	fmt.Printf("Ciphertext: %x\n", ciphertext)

	// 解密
	mode = cipher.NewCBCDecrypter(block, iv)
	decrypted := make([]byte, len(paddedText))
	mode.CryptBlocks(decrypted, ciphertext[aes.BlockSize:])

	// 去除 PKCS#7 填充
	pad := int(decrypted[len(decrypted)-1])
	decrypted = decrypted[:len(decrypted)-pad]

	fmt.Printf("Decrypted: %s\n", decrypted)
}

RSA 加密

RSA 是一种非对称加密算法，使用一对公钥和私钥进行加密和解密：

package main

import (
	"crypto/rand"
	"crypto/rsa"
	"crypto/sha256"
	"fmt"
)

func main() {
	// 生成密钥对，私钥和公钥
	privateKey, err := rsa.GenerateKey(rand.Reader, 2048)
	if err != nil {
		fmt.Println("Private key generation failed:", err)
		return
	}
	publicKey := &privateKey.PublicKey

	// 要加密的字节切片
	message := []byte("hello world")

	// 使用 OAEP 模式和 SHA-256 哈希函数进行加密
	encryptedMessage, err := rsa.EncryptOAEP(
		sha256.New(),
		rand.Reader,
		publicKey,
		message,
		nil) // 标签通常为空
	if err != nil {
		fmt.Println("Encryption failed:", err)
		return
	}
	fmt.Printf("Encrypted: %x\n", encryptedMessage)

	// 解密消息
	decryptedMessage, err := rsa.DecryptOAEP(
		sha256.New(),
		rand.Reader,
		privateKey,
		encryptedMessage,
		nil)
	if err != nil {
		fmt.Println("Decryption failed:", err)
		return
	}
	fmt.Printf("Decrypted: %s\n", decryptedMessage)
}
