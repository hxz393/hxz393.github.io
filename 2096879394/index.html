Go 语言类型：映射

发表于 2024-03-01 更新于 2024-07-23 分类于 Go ， 2.类型系统 阅读次数： 本文字数： 1.7k 阅读时长 ≈ 6 分钟

映射定义

映射（Map）是用于储存一系列无序的键值对（Key-value Pair）的数据结构。映射通过把键映射到值来进行访问，能够快速检索数据。映射有时也叫做字典、哈希表、散列和关联数组。

声明映射

以下是声明映射的常用方法：

  • var 映射名 map[键类型]值类型：标准映射变量声明格式。

  • 映射名 := make(map[键类型]值类型, 容量)：用 make 函数来创建映射，并分配内存。容量参数可选。

  • 映射名 := map[键类型]值类型{键1: 值1, 键2: 值2...}：创建映射并使用字面量初始化。

下面是声明映射示例代码：

package main

import "fmt"

func main() {
	// 使用 var 关键字声明一个映射来存储员工薪资
	var employeeSalaries map[string]int
	// 在使用之前必须通过 make() 函数初始化映射
	employeeSalaries = make(map[string]int)
	employeeSalaries["Alice"] = 68000
	fmt.Println("Alice 的薪资：", employeeSalaries["Alice"])

	// 使用 make() 函数直接创建并初始化员工薪资映射
	salariesByEmployee := make(map[string]int)
	salariesByEmployee["Bob"] = 72000
	fmt.Println("Bob 的薪资：", salariesByEmployee["Bob"])

	// 使用字面值初始化员工薪资映射。如果采用多行来初始化，最后一个元素后面必须加逗号
	departmentSalaries := map[string]int{"Carol": 85000, "Dan": 90000}
	fmt.Println("各员工薪资：", departmentSalaries)
}

映射值类型没有限制，但是键不能使用引用类型，例如切片、函数和映射：

package main

import "fmt"

func main() {
	// 尝试使用切片作为映射的键会引发编译错误
	//map[[]int]string{} // 错误：invalid map key type []int

	// 正确的方式：使用固定长度数组作为键
	salaryKeysByArray := map[[3]int]string{}
	salaryKeysByArray[[3]int{1, 2, 3}] = "123"
	fmt.Println("使用数组作为键的映射：", salaryKeysByArray)
}

nil 映射

值为 nil 的空映射，由于没有分配内存，不能用于储存键值对。在使用前必须用 make 初始化，否则在尝试添加键值对时会引发运行时错误：

package main

import "fmt"

func main() {
	// 声明一个 nil 映射
	var m map[string]int

	// 可以读取 nil 映射的键值
	value := m["someKey"]
	fmt.Println("读取不存在的键值:", value) // 输出：读取不存在的键值: 0

	// 尝试向 nil 映射添加键值对会导致恐慌
	//m["someKey"] = 1 // 报错：assignment to entry in nil map

	// 正确的初始化方式，使用 make 函数
	m = make(map[string]int)
	m["someKey"] = 1
	fmt.Println("添加键值后的映射:", m) // 输出：添加键值后的映射: map[someKey:1]
}

nil 映射通常用于只需要执行读取的场合，替代空映射传递给函数。

读取键值对

从映射取值可以获得值以及一个键存在的布尔值标志，即使键不存在也会返回映射值类型的默认零值：

package main

import "fmt"

func main() {
	employeeSalaries := map[string]int{"Alice": 68000, "Bob": 72000}

	// 对返回分别赋值
	salary, exists := employeeSalaries["Charlie"]
	if exists {
		fmt.Println("Charlie 的薪资为：", salary)
	} else {
		fmt.Println("未找到 Charlie 的薪资记录。", exists) // exists 值为 false
		fmt.Println("试图访问不存在的键导致返回的零值：", salary)  // 此处 salary 为 0
	}
}

最佳实践中读取键值对时，应该始终检查键值对是否存在于映射中。

遍历映射

可以使用 for 和 range 来迭代映射里的键和值。Go 在迭代映射时不保证键的顺序，这是故意为之，以防程序依赖于键的顺序：

package main

import "fmt"

func main() {
	employeeDepartment := map[string]int{"Alice": 101, "Bob": 102, "Charlie": 104, "David": 103}

	// 使用 for 和 range 遍历映射，每次迭代输出的顺序可能不同
	fmt.Println("员工部门列表：")
	for name, department := range employeeDepartment {
		fmt.Printf("%s 在部门编号：%d\n", name, department)
	}
	// 使用 len 获取映射的键值对数量
	fmt.Println("员工数量：", len(employeeDepartment))
}

新增修改键值对

修改映射的内容可以通过简单地赋值来实现。如果键不存在于映射中，赋值操作会添加一个新的键值对；如果键已存在，赋值操作会更新该键对应的值：

package main

import "fmt"

func main() {
	employeeSalaries := map[string]int{"Alice": 68000, "Bob": 72000}
	fmt.Println("初始员工薪资映射：", employeeSalaries)

	// 添加新员工的薪资
	employeeSalaries["Charlie"] = 90000
	fmt.Println("添加 Charlie 后的员工薪资映射：", employeeSalaries)

	// 更新已有员工的薪资
	employeeSalaries["Alice"] = 71000
	fmt.Println("更新 Alice 薪资后的员工薪资映射：", employeeSalaries)
}

删除键值对

Go 语言提供内置 delete 函数用于删除映射中的元素。如果尝试删除的键在映射中不存在，这个操作会被安全地忽略，不会引发错误或异常：

package main

import "fmt"

func main() {
	employeeSalaries := map[string]int{"Alice": 7000, "Bob": 8000, "Charlie": 9000}
	fmt.Println("初始员工薪资映射：", employeeSalaries)

	// 删除存在的键
	delete(employeeSalaries, "Bob")
	fmt.Println("删除 Bob 后的员工薪资映射：", employeeSalaries)

	// 尝试删除不存在的键
	delete(employeeSalaries, "David")
	fmt.Println("尝试删除不存在的 David 后的员工薪资映射：", employeeSalaries)

	// 再次验证Bob是否已被删除
	if salary, exists := employeeSalaries["Bob"]; !exists {
		fmt.Println("Bob 的薪资记录已被删除，目前不存在。")
	} else {
		fmt.Println("Bob 的薪资是：", salary)
	}
}

复制映射

和数组类似，映射是引用类型，所以在函数中对传递的映射进行修改，会作用于所有被传递映射的引用。为了避免函数副作用，可以在函数内部创建映射的副本，并对该副本进行操作：

package main

import "fmt"

func modifyMap(originalMap map[string]int) map[string]int {
	// 创建一个新的映射副本
	mapCopy := make(map[string]int)
	for key, value := range originalMap {
		mapCopy[key] = value
	}

	// 在副本上进行修改
	mapCopy["newKey"] = 100

	// 返回修改后的映射副本
	return mapCopy
}

func main() {
	myMap := map[string]int{"a": 1, "b": 2}
	modifiedMap := modifyMap(myMap)

	fmt.Println("原始映射：", myMap)
	fmt.Println("修改后的映射：", modifiedMap)
}

复制映射需要创建一个新的空映射，并通过遍历原始映射的所有键值对来填充这个新映射，这样两者之间的数据才完全独立。

模拟集合

Go 语言没有集合类型，可利用映射中键的唯一性，来模拟集合去重功能。将需要放入集合的值作为映射的键，使用布尔值作为值的类型，把值依次存入映射：

package main

import "fmt"

func main() {
	// 建立一个映射来模拟集合
	set := map[int]bool{}
	// 要存入集合的值列表
	vals := [...]int{1, 2, 3, 2, 1}

	// 利用键的唯一性，将值写入到映射中的键
	for _, v := range vals {
		set[v] = true
	}

	// 通过查询映射的键值来判断状态
	fmt.Println(set[1])  // 值已存入，输出：true
	fmt.Println(set[10]) // 查询的值没在映射中，输出布尔型零值：false

	// 创建一个切片来保存结果
	intSet := []int{}
	for k, _ := range set {
		intSet = append(intSet, k)
	}
	fmt.Println(intSet) // 输出: [1 2 3]
}
