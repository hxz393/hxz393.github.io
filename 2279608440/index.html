使用 Go 语言处理 JSON

发表于 2024-06-27 更新于 2024-07-23 分类于 Go ， 4.基本应用 阅读次数： 本文字数： 1.4k 阅读时长 ≈ 5 分钟

JSON 介绍

JSON（JavaScript Object Notation）全称 JavaScript 对象表示法，是一种轻量级的储存和交换数据格式。JSON 使用文本格式，可以用键值对和数组的方式表示数据。JSON 在网络应用中非常流行，已经成为互联网储存和交换数据的事实标准，大多数编程语言都支持 JSON 数据编码和解码。

JSON 数据格式包括：

  • 对象（Object）： 由花括号 {} 包围，里面包含一系列键值对。每个键是一个字符串，后面跟一个冒号，然后是值。
  • 数组（Array）： 由方括号 [] 包围，里面包含一系列值，值之间用逗号分隔。
  • 值（Value）： 可以是字符串、数字、布尔值、数组、对象或者 null。

例如一个表示用户的 JSON 对象：

{
  "name": "张三",
  "age": 28,
  "isStudent": false,
  "skills": ["Java", "Python", "JavaScript"],
  "address": {
    "street": "科技路1号",
    "city": "北京",
    "country": "中国"
  }
}

上面代码中，值数据类型分别为：字符串、数字、布尔值、数组和另一个对象。JSON 中的数据类型和 Go 语言中的数据类型一般对应关系如下：

JSON Go
Boolean bool
Number float64
String string
Array []interface{}
Object map[string]interface{}
Null nil

当然也可以自定义类型对应关系，例如把字符串格式的时间映射到 time.Time 类型，但需要注意如果数据类型不匹配，转换的时候可能会出错。

JSON 格式

在 Go 语言中，可以使用自定义结构体加字段标签来储存 JSON 格式数据：

package main

import  "fmt"

// User 结构体中元素数据类型一一对应 JSON 格式
type User struct {
	Name      string   `json:"name"`
	Age       int      `json:"age"`
	IsStudent bool     `json:"isStudent"`
	Skills    []string `json:"skills"`
	Address   Address  `json:"address"`
}
// Address 结构体作为 User 结构体内嵌结构体，以便单独赋值
type Address struct {
	Street  string `json:"street"`
	City    string `json:"city"`
	Country string `json:"country"`
}

func main() {
	// 使用字面量来初始化结构体
	address := Address{
		Street:  "科技路1号",
		City:    "北京",
		Country: "中国",
	}
	user := User{
		Name:      "张三",
		Age:       28,
		IsStudent: false,
		Skills:    []string{"Java", "Python", "JavaScript"},
		Address:   address,
	}
	
	fmt.Println(user)
}

如果使用 GoLand，向编辑区粘贴 JSON 格式内容，编辑器可以自动生成对应的结构体数据格式。

序列化

在 encoding/json 包中提供了函数 Marshal，用于将结构体变量转换为 JSON 格式：

package main

import (
	"encoding/json"
	"fmt"
)

func main() {
	// 使用匿名结构体
    // JSON 键名后的 omitempty 代表允许忽略空值
	user := struct {
		Name      string `json:"name"`
		Age       int    `json:"age"`
		IsStudent bool   `json:"isStudent,omitempty"` 
	}{"张三", 28, false}
	fmt.Println(user) // 输出：{张三 28 false}

	// 修改元素值，以区分输出结果
	user.Age = 18
	// 将结构体变量 user 转为 JSON 格式
	// 由于 IsStudent 为 false 空值，且包含 omitempty 选项，转换时被忽略了
	data, err := json.Marshal(user)
	if err != nil {
		fmt.Println(err)
		return
	}
	fmt.Println(string(data)) // 输出：{"name":"张三","age":18}
}

反序列化

解码 JSON 格式数据，使用 encoding/json 包中提供的函数 Unmarshal。函数参数接受 JSON 格式变量和一个接口变量，可以理解为将输入的 JSON 数据按照接口的数据结构储存：

package main

import (
	"encoding/json"
	"fmt"
)

// User 结构体用于储存对应的 JSON 数据
type User struct {
	Name      string `json:"userName"` // 故意写错，这个字段会匹配不上
	Age       int    `json:"Age"`      // 故意混淆大小写，但不影响映射关系
	IsStudent bool   `json:"isStudent,omitempty"`
}

func main() {
	// 必须声明一个 User 结构体变量
	var user User
	// JSON 格式为字节切片
	data := []byte(`{"name":"张三","age":18, "isStudent":false}`)

	// 将 JSON 格式变量 data 反序列化，赋值给变量 user
	// 其实 name 字段会转换失败，但不会报错
	err := json.Unmarshal(data, &user)
	if err != nil {
		fmt.Println(err)
		return
	}
	fmt.Printf("%+#v\n", user) // 输出：main.User{Name:"", Age:18, IsStudent:false}
}

嵌套数据不能被编码转换，如果不希望导出结构体的某字段，可以在标记中写成 json:"-"。

解码网络数据

如果是通过网络获取的 JSON 数据，可以直接使用 NewDecoder 配合 Decoder 函数来解码到结构体：

package main

import (
	"encoding/json"
	"fmt"
	"net/http"
)

// User 结构体只取两个字段
type User struct {
	Login string `json:"login"`
	Id    int    `json:"id"`
}

func main() {
	var user User
	r, _ := http.Get("https://api.github.com/users/hxz393")
	defer r.Body.Close()

	_ = json.NewDecoder(r.Body).Decode(&user) // 直接解码响应体后赋值给 user
	fmt.Printf("%+#v\n", user)                // 输出：main.User{Login:"hxz393", Id:5063578}
}

解码文件数据

如果是通过文件读取的 JSON 数据，流程像下面这样：

package main

import (
	"encoding/json"
	"fmt"
	"os"
)

type Verified struct {
	Description   string `json:"description"`
	SignedContent struct {
		Payload    string `json:"payload"`
		Signatures []struct {
			Header struct {
				Kid string `json:"kid"`
			} `json:"header"`
			Protected string `json:"protected"`
			Signature string `json:"signature"`
		} `json:"signatures"`
	} `json:"signed_content"`
}

func main() {
	// 读取文件内容
	bytes, err := os.ReadFile("verified_contents.json")
	if err != nil {
		fmt.Println("Error reading file:", err)
		return
	}

	// 反序列化 JSON 数据到 Verified 结构体的切片中
	var verified []Verified
	if err := json.Unmarshal(bytes, &verified); err != nil {
		fmt.Println("Error unmarshaling JSON:", err)
		return
	}

	fmt.Printf("Verified: %+#v\n", verified)
}

第三方库

自带的 encoding/json 包使用反射来实现类型到 JSON 的转换，可能导致性能不好。第三方包 https://github.com/valyala/fastjson 不使用反射，直接解析 JSON 数据到特定的结构，在解析大型文件或高频率调用解析功能时性能差别明显：

package main

import (
	"encoding/json"
	"testing"

	"github.com/valyala/fastjson"
)

// 定义一个结构体用于测试
type Person struct {
	Name    string `json:"name"`
	Age     int    `json:"age"`
	Country string `json:"country"`
}

// 测试数据
var jsonData = `{"name":"John Doe","age":30,"country":"USA"}`

// 使用 encoding/json 解析JSON
func BenchmarkEncodingJSON(b *testing.B) {
	var p Person
	for i := 0; i < b.N; i++ {
		if err := json.Unmarshal([]byte(jsonData), &p); err != nil {
			b.Fatal(err)
		}
	}
}

// 使用 fastjson 解析JSON
func BenchmarkFastJSON(b *testing.B) {
	var p fastjson.Parser
	for i := 0; i < b.N; i++ {
		v, err := p.Parse(jsonData)
		if err != nil {
			b.Fatal(err)
		}
		_ = v.GetStringBytes("name")
		_ = v.GetInt("age")
		_ = v.GetStringBytes("country")
	}
}
