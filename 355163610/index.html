Go 语言类型：字符串型

发表于 2024-01-19 更新于 2024-07-23 分类于 Go ， 2.类型系统 阅读次数： 本文字数： 4.9k 阅读时长 ≈ 16 分钟

字符串定义

一个字符串（String）在 Go 语言中是一个不可改变的 UTF-8 编码的字节序列。字符串可以理解为不可变的字节数组，数组中的每个元素是一个字节，而一个完整的 Unicode 字符可能由一个或多个字节组成。

创建字符串变量

常见的创建字符串变量方式有直接赋值和从字节或字符切片转换。

直接赋值字面量

直接赋值字符串字面量用双引号「"」括起来，可以包含转义字符：

package main

import "fmt"

func main() {
	str := "Hello\nWorld!" // 带转义字符换行
	fmt.Println(str)
}

原生字符串字面量

原生字符串字面量使用反引号「`」括起来，支持多行字符串，里面不会解析转义字符，多用于 HTML 和正则表达式：

package main

import "fmt"

func main() {
	str := `第一行

第三行，不转换转义字符 \n
  第四行，前面带两空格`
	fmt.Println(str)
}

从字节或字符切片构造

从字节或字符切片创建字符串变量的方式一样：

package main

import "fmt"

func main() {
	// 字节切片
	bytes := []byte{72, 101, 108, 108, 111}
	str := string(bytes)
	fmt.Println(str)

	// 字符切片
	runes := []rune{'你', '好', '世', '界'}
	str = string(runes)
	fmt.Println(str)
}

从字符串切片创建

字符串不能通过索引修改某个字节的值，基于字符串创建的切片还是字符串，这些切片同样不能被修改：

package main

import "fmt"

func main() {
	str := "hello world"
	//str[0] = 'H'            // 直接修改字符串报错：cannot assign to str[0]
	singleWord := str[:5]   // 对字符串切片结果仍然是字符串类型
	//singleWord[1] = "11"    // 报错：cannot assign to singleWord[1]
	fmt.Println(singleWord) // 输出："hello"
}

字符串格式化

字符串的格式化是通过 fmt 包中的函数（如 fmt.Printf, fmt.Sprintf 等）实现。动词形式是 %ML，百分号视为格式化动词的开始，其中 M 表示一个或者多个可选的动词修饰符，而 L 表示一个特定的格式动词字符。有些格式化动词可以接收多个修饰符。

下面是一些常用的字符串格式化动词及其含义的表格：

作用对象 动词 含义
所有类型 %v 使用默认格式输出，或是用其类型的 String 方法输出。在打印结构体时，会添加字段名。和 # 修饰符搭配使用，能打印出不可见字符，例如回车、制表符等
所有类型 %T 打印值的类型
所有类型 %% 打印字面量上的百分号
布尔型 %t 打印布尔值
整型 %b 打印二进制格式，将整数格式化为二进制的表达方式
整型 %c 打印 Unicode 字符格式
整型 %d 打印十进制格式
整型 %o 打印八进制格式
整型 %q 该值对应的单引号围绕的 Go 语法字符串字面值，适用于字符，例如：'\x01'
整型或字符串 %x 打印十六进制格式，字母形式为小写 a-f
整型或字符串 %X 打印十六进制格式，字母形式为大写 A-F
整型 %U 打印 Unicode 格式，默认值为 4 个数字字符，例如 ：U+554A
指针 %p 以十六进制表示的指针（内存地址），前缀为 0x，字母形式为小写 a-f，例如：0xc0420e8058
字符串或切片 %s 使用 UTF-8 格式，直接打印字符串或切片
字符串 %q 打印字符串同时自动加双引号
浮点型或复数 %f 浮点数表示，可带修饰符。例如指定宽度为 10，精度为 3：%10.3f
浮点型或复数 %e 科学计数法 e 表示，例如：-1234.456e+78
浮点型或复数 %E 科学计数法 E 表示，例如：-1234.456E+78
浮点型或复数 %g 根据情况选择 %e 或 %f 以产生更紧凑的（无多余零的）输出
浮点型或复数 %G 类似于 %g，但是使用 %E 或 %f
修饰符 空格 在输出数字前面，对正数增加空格，对复数加上减号。输出十六进制格式时，在每个字节前加上空格。例如：fmt.Printf("% x\n", "2000") 的结果为：32 30 30 30
修饰符 + 打印数值的正负号，例如：fmt.Printf("%+d", 111) 将会输出 +111；打印结构体的字段名，例如：fmt.Printf("%+v", *bb*) 将会输出：{a1: a2:0}；打印字符串中 ASCII 字符，非 ASCII 字符输出转义字符，例如：fmt.Printf("%+q", "z啊z") 将输出："z\u554az"
修饰符 - 通常与宽度指示符一起使用，来指定输出的最小宽度。也就是将值向左对齐，使用空格填充右侧空缺。例如：fmt.Printf("#%-3s#%-4s#\n", "aa", "bb") 将输出：#aa #bb #
修饰符 0 同样与宽度指示符一起使用，使用前置 0 代替空格填补空缺。例如：fmt.Printf("#%03s#%04s#\n", "aa", "bb") 将输出：#0aa#00bb#
修饰符 # 切换格式，例如打印出整数 111 的十六进制格式 0x6f：fmt.Printf("%#x", 111)
修饰符 n.m 宽度指示符，n 和 m 都是整数。n 代表输出的最小宽度，实际输出太少则会以空格填充。对于数字， m 代表小数点后保留位数。例如：fmt.Printf("%06.2f", 22.333) 将输出：022.33。对于字符串，m 限制字符串输出最大长度，实际输出超过部分会被从后截断。例如：fmt.Printf("%06.5s", "aabbcc") 将输出：0aabbc。n 或 .m 都可以被省略

输出函数 fmt.Print 和 fmt.Println 都是使用 %v 作为默认格式，因此可以打印任何内容。

字符串转换

由于字符串不能修改，因此在实际中，字符串通常需要转为字节或字符切片类型来处理：

package main

import "fmt"

func main() {
	str := "Hello, 世界!"

	// 转为字节切片
	bytes := []byte(str)
	fmt.Println("字节数组:", bytes)
	fmt.Printf("字节输出: ")
	for _, b := range bytes {
		fmt.Printf("%x ", b)
	}
	fmt.Println()

	// 转为字符切片
	runes := []rune(str)
	fmt.Println("字符数组:", runes)
	fmt.Printf("字符输出: ")
	for _, r := range runes {
		fmt.Printf("%q ", r)
	}
	fmt.Println()
}

字符串索引

虽然字符串不能修改，但可以通过索引读取字符串的值。不可以获取字符串中某字节的地址：

package main

import "fmt"

func main() {
	str := "Hello, 世界!"

	// 访问字符串的第一个字节
	firstByte := str[0]
	fmt.Printf("字符串 '%s' 的第一个字节是 '%c'，字节值为 %d\n", str, firstByte, firstByte)

	// 试图访问字符串中的 Unicode 字符的某个字节，访问到多字节字符的中间字节
	byteOfMultiByteChar := str[7]
	fmt.Printf("在字符串 '%s' 的索引 7 处的字节是 '%c'，字节值为 %d\n", str, byteOfMultiByteChar, byteOfMultiByteChar)

	// 处理多字节字符先转为字符切片后再索引
	runeAtPosition := []rune(str)[7]
	fmt.Printf("在字符串 '%s' 中位置 7 的字符是 '%c'，字符值为 %U\n", str, runeAtPosition, runeAtPosition)

	// 对字符串中字节取址和设值都将报错
	//fmt.Println(&str[0]) // 报错：cannot take the address of str[0]
	//str[0] = 'w'         // 报错：cannot assign to str[0]
}

字符串遍历

字符串遍历，是按照下标顺序读取字符串中的每个字符或字节，输出的为字符编码数字：

package main

import "fmt"

func main() {
	// 定义一个带中文字的字符串
	str := "cn=中文"

	// 分别输出 5 个字符和对应编码
	fmt.Println("以字符方式遍历：")
	for _, v := range str {
		fmt.Printf("字符：%c 的 Unicode 编码为：%v \n", v, v)
	}

	// 分别输出 9 个整数。中文字在 UTF-8 编码中占 3 个字节
	fmt.Println("以字节方式遍历：")
	for i := 0; i < len(str); i++ {
		fmt.Printf("第 %d 个字节的编码为：%v \n", i, str[i])
	}
}

字符串修改

在 Go 语言中一旦字符串被创建，其内容就不能更改。不过可以先将字符串转换为字符切片，进行修改后再转换回字符串：

package main

import "fmt"

func main() {
	original := "Hello, World!"
	// 先转为字符数组
	runes := []rune(original)
	runes[7] = 'G' // 'W' 改为 'G'
	// 保留前 8 个字符，并将之后的字符替换为 "o!"
	runes = append(runes[:8], 'o', '!')

	modified := string(runes)
	fmt.Println(modified) // 输出 "Hello, Go!"
}

字符串拼接

可以使用加号「+」来合并字符串的片段，创建一个新的字符串：

package main

import "fmt"

func main() {
	original := "Hello, World!"

	modified := original[:7] + "Go!" // 两个字符串直接拼接
	fmt.Println(modified)           // 输出 "Hello, Go!"

	original += modified  // 采用追加操作符来级联追加
	fmt.Println(original) // 输出 "Hello, World!Hello, Go!"
}

当需要拼接的字符串较大或拼接很频繁时，推荐使用字节缓冲的方式来进行高效的字符串连接：

package main

import (
	"bytes"
	"fmt"
	"reflect"
)

func main() {
	// 创建一个空的 bytes.Buffer 类型变量
	var buffer bytes.Buffer
	// 使用 WriteString() 方法，将字符串写入到缓冲中
	buffer.WriteString("Hello")
	buffer.WriteString(", ")
	buffer.WriteString("World!")
	buffer.Truncate(7) // 只保留前 7 个字节
	buffer.WriteString("Go!")

	fmt.Println(buffer)                 // 输出变量 buffer 完整信息
	fmt.Println(buffer.String())        // 转为字符串，输出 "Hello, Go!"
	fmt.Println(reflect.TypeOf(buffer)) // 输出类型为：bytes.Buffer
}

字符串统计

将字符串转为切片后，可以用 len 函数统计字符串大小和字数：

package main

import (
	"fmt"
	"unicode/utf8"
)

func main() {
	s := "Hello, 世界!"
	fmt.Printf("%d\n", len(s))             // 统计字符串字节大小为 14
	fmt.Printf("%d\n", len([]rune(s)))     // 转换成字符切片后，统计得到字符数 10
	fmt.Println(utf8.RuneCountInString(s)) // 和上面同样的结果，符号和字一视同仁
}

要统计指定字符串出现的次数，可以用 strings.Count 计算：

package main

import (
	"fmt"
	"strings"
)

func main() {
	s := "我叫啊佩，你别叫我啊佩"
	fmt.Println(strings.Count(s, "啊佩")) // 返回次数 2
}

字符串比较

字符串可以使用 == 和大小比较符号进行比较。比较大小判断依据是第一个字节编码数字大小：

package main

import "fmt"

func main() {
	str1 := "Hello"
	str2 := "hello"

	// 使用 < 或 > 比较字符串的字典序
	fmt.Println("str1 大于 str2:", str1 > str2)
	fmt.Println("str1:", str1[0], "str2:", str2[0])
}

如果比较时需要忽略字母大小写，可以使用 strings.EqualFold 函数：

package main

import (
	"fmt"
	"strings"
)

func main() {
	str1 := "Hello"
	str2 := "hello"

	// 使用 strings.EqualFold() 进行不区分大小写的比较
	fmt.Println("str1 等于 str2:", strings.EqualFold(str1, str2))
}

字符串搜索

一般使用 Go 语言标准库中的 strings 库来进行字符串搜索操作。

包含

判断字符串所在位置的三种方法：

package main

import (
	"fmt"
	"strings"
)

func main() {
	s := "The quick brown fox jumps over the lazy dog"
	fmt.Println(strings.HasSuffix(s, " dog"))       // 搜索后缀，返回 true
	fmt.Println(strings.HasPrefix(s, "The quick ")) // 搜索前缀，返回 true
	fmt.Println(strings.Contains(s, "fox"))         // 搜索整个字符串，返回 true
}

此外还有一个 ContainsAny 函数，检查字符串中是否包含指定字符集中的任意字符：

package main

import (
	"fmt"
	"strings"
)

func main() {
	s := "abcdef"
	fmt.Println(strings.Contains(s, "ag"))    // 返回 false
	fmt.Println(strings.ContainsAny(s, "ag")) // 由于 s 在字符串中，返回 true
	fmt.Println(strings.Contains(s, ""))      // 返回 true
	fmt.Println(strings.ContainsAny(s, ""))   // 返回 false
}

注意下搜索子字符串为空的情况：空字符串被视为所有字符串的子串，因此调用 strings.Contains 检查是否包含空字符串结果都是 true。而 strings.ContainsAny 函数检查给定的字符集中的任意字符是否出现在目标字符中，空字符串视为没有字符要检查，因此返回 false。

索引

除了返回布尔值，还能使用 strings.Index 函数来返回子字符串第一次出现位置索引。针对非 ASCII 编码字符用 strings.IndexRune 函数来准确定位字符：

package main

import (
	"fmt"
	"strings"
)

func main() {
	s := "abc啊佩def"
	fmt.Println(strings.Index(s, "abc"))   // 索引开始于 0
	fmt.Println(strings.Index(s, "def"))   // 1 个中文字占了 3 个索引，所以返回 9
	fmt.Println(strings.IndexRune(s, '佩')) // 通过字符搜索，返回 6
	fmt.Println(strings.Index(s, "ag"))    // 没找到返回 -1
}

假如要返回字符串最后一次出现位置索引，使用 strings.LastIndex 函数：

package main

import (
	"fmt"
	"strings"
)

func main() {
	s := "学习书籍，学习资源，学习笔记"
	searchIndex := strings.LastIndex(s, "学习")
	fmt.Println(searchIndex)     // 索引返回 30，也就是 10 个中文字符后的位置
	fmt.Println(s[searchIndex:]) //  对原字符切片，返回 "学习笔记"
}

字符串替换

简单的字符串替换可以使用 strings.Replace 函数来操作。函数接收 4 个参数：原字符串、要被替换的子字符串、替换后的新字符串、替换操作次数，返回新字符串。如果要全部替换，次数传入 -1：

package main

import (
	"fmt"
	"strings"
)

func main() {
	originalString := "Welcome to Go programming. Go is fun."

	// 替换第一次出现的 "Go" 为 "Golang"
	resultOne := strings.Replace(originalString, "Go", "Golang", 1)
	fmt.Println("替换一次后:", resultOne) // 输出：Welcome to Golang programming. Go is fun.

	// 替换所有的 "Go" 为 "Golang"
	resultAll := strings.Replace(originalString, "Go", "Golang", -1)
	fmt.Println("全部替换后:", resultAll) // 输出：Welcome to Golang programming. Golang is fun.
}

另一种字符串替换方式是使用 strings.Map 函数，它需要接受一个签名为 func(rune) rune 的映射函数和要处理的字符串，对字符串中每个字符都调用映射函数处理。如果映射函数返回负数，原字符会被删除：

package main

import (
	"fmt"
	"strings"
	"unicode"
)

func main() {
	originalString := "Hello, World! 123 Go!"

	// 转换所有字符为大写
	upperFunc := func(r rune) rune {
		return unicode.ToUpper(r)
	}
	upperString := strings.Map(upperFunc, originalString)
	fmt.Println("全部转大写:", upperString) // 输出：HELLO, WORLD! 123 GO!

	// 删除所有的数字
	removeDigits := func(r rune) rune {
		if unicode.IsDigit(r) {
			return -1 // 返回 -1 表示删除此字符
		}
		return r
	}
	noDigits := strings.Map(removeDigits, originalString)
	fmt.Println("删除数字后:", noDigits) // 输出：Hello, World!  Go!
}

这种方法非常适合执行字符级别的转换，例如大小写转换、字符过滤等。

大小写转换

使用 strings 包中的 ToLower 和 ToUpper 函数，能对字符串中的所有字符进行大小写转换：

package main

import (
	"fmt"
	"strings"
)

func main() {
	originalString := "Guten Tag! ¿Cómo estás?"

	// 转换为大写
	upper := strings.ToUpper(originalString)
	fmt.Println("转换为大写:", upper) // 输出：GUTEN TAG! ¿CÓMO ESTÁS?

	// 转换为小写
	lower := strings.ToLower(originalString)
	fmt.Println("转换为小写:", lower) // 输出：guten tag! ¿cómo estás?
}

如果需要更复杂的字符处理逻辑，或者对字符进行单独的处理，可以使用 unicode.ToUpper 和 unicode.ToLower 通过 strings.Map 的方式在转换过程中插入更多的逻辑。

字符串修剪

字符串修剪指去除一些不需要的字符，例如字符串首尾的空格：

package main

import (
	"fmt"
	"strings"
)

func main() {
	fmt.Println(strings.Trim("?a?b?c?", "?a"))   // 剔除首尾的 ? 号和 a，输出 b?c
	fmt.Println(strings.TrimLeft("??a??", "?"))  // 只剔除左边，输出 a??
	fmt.Println(strings.TrimRight("??a??", "?")) // 只剔除右边，输出 ??a
	fmt.Println(strings.TrimSpace(" a\tb  \n"))  // 剔除首尾空格、换行、制表符等
	fmt.Println(strings.Trim("我想慢慢吃慢慢品", "慢我想")) // 剪切集 慢我想 中的任何字符，最后输出 吃慢慢品
}

字符串重复

可以使用 strings.Repeat 函数来重复字符串指定次数：

package main

import (
	"fmt"
	"strings"
)

func main() {
	s := "SOS! "
	fmt.Println(strings.Repeat(s, 3)) // 输出：SOS! SOS! SOS!
}

字符串分割

用 strings.Split 函数分割字符串，函数接受两个参数，目标字符串和分割字符串，根据指定的分隔符将字符串分割成若干部分，结果返回一个切片：

package main

import (
	"fmt"
	"strings"
)

func main() {
	// 用逗号分割 "hello, world" 得到两个元素的切片
	a := strings.Split("hello,   world", ",")
	fmt.Printf("%q\n", a) // 打印切片内容 ["hello" "   world"]
	fmt.Printf("%T\n", a) // 打印类型为 []string
	for _, a := range a {
		fmt.Println(a) // 遍历输出切片元素 hello   world
	}
}

另外有 strings.SplitN 函数来限定分割次数，传入第三个整数参数作为分割次数：

package main

import (
	"fmt"
	"strings"
)

func main() {
	str := "a,b,c,d,e"
	parts := strings.SplitN(str, ",", 3)
	fmt.Println(parts) // 输出：[a b c,d,e]，也就是分为了三个元素：a、b 和 c,d,e
}

此外还有一个常用的分割函数 strings.Fields，用于按空白字符分割字符串，函数可以自动识别和处理所有空白字符。

字符串插入

除了使用字符串拼接方法外，还可以利用 string 包中的函数 strings.Join 来来间接实现字符串的插入。

先使用 strings.Fields 函数把字符串转为字符串切片，然后使用 strings.Join 在元素之间插入新的分隔符来重新组合成新的字符串：

package main

import (
	"fmt"
	"strings"
)

func main() {
	a := "This is a test with function strings"

	// 使用 strings.Fields 将字符串分割成单词切片
	aSli := strings.Fields(a)
	fmt.Println("输出切片内容：", aSli) // 输出切片内容：[This is a test with function strings]

	// 遍历切片，分别打印出每个单词
	for _, v := range aSli {
		fmt.Println(v) // 分别打印出以空格分隔的每个单词
	}

	// 使用 strings.Join 将单词切片连接成一个新字符串，单词之间用"?"分隔
	b := strings.Join(aSli, "?")
	fmt.Printf("输出连接后的字符串：%s\n", b) // 输出：This?is?a?test?with?function?strings
}

字符串转换

可以使用 strconv 包完成字符串和基本数据类型之间转换。Format 开头函数用于将基本数据类型转为字符串，几乎所有类型都能被转为字符串；Parse 开头函数用于将字符串转为其他基本数据类型：

package main

import (
	"fmt"
	"strconv"
)

func main() {
	var s string = "true"
	b, _ := strconv.ParseBool(s)
	fmt.Printf("Type: %T, Value: %v\n", b, b) // 输出布尔类型值：Type: bool, Value: true
	c := strconv.FormatBool(b)
	fmt.Printf("Type: %T, Value: %v\n", c, c+" string") // 输出字符串类型值：Type: string, Value: true string
	d := 11
	e := strconv.Itoa(d)
	fmt.Printf("Type: %T, Value: %v\n", e, e) // 输出字符串类型值：Type: string, Value: 11
}

上面代码把字符串 true 转为布尔值 true。如果输入字符串不是有效的布尔值，则转为布尔值后为 false。

转换整型和浮点型数据到字符串可以使用 strconv.Itoa 和 strconv.FormatFloat 等函数。对应的反向转换函数有 strconv.Atoi 和 strconv.ParseFloat 等函数。

通常将字符串转为其他类型可能会失败并返回错误，因此函数第二个返回值为报错内容：

package main

import (
	"fmt"
	"strconv"
)

func main() {
	// 试图将 3 进制数 14 转为 10 进制
	s, err := strconv.ParseInt("14", 3, 32)
	fmt.Println(err)                          // 报语法错误：strconv.ParseInt: parsing "14": invalid syntax
	fmt.Printf("Type: %T, Value: %v\n", s, s) // 输出默认零值：Type: int64, Value: 0
}

字符串编码

在 Go 语言中，unicode/utf8 和 unicode 包提供了不同但互补的功能集，用于处理 Unicode 字符和 UTF-8 编码。

unicode/utf8 包提供下面功能：

  • 编码和解码 UTF-8：unicode/utf8 包专注于 UTF-8 编码，它提供了函数来编码和解码单个 Unicode 字符。例如将 Unicode 字符编码为 UTF-8 序列并返回编码的长度：EncodeRune(p []byte, r rune) int
  • 检查字节的 UTF-8 编码：提供函数来检查一个字节序列是否是有效的 UTF-8 编码，并确定一个 UTF-8 编码字符的长度。例如返回字符串中的字符数量：RuneCountInString(s string) int
  • 字符计数和范围遍历：允许计算字符串中的字符数量（而不是字节），并在 UTF-8 编码的字符串中遍历每个 Unicode 字符。例如从字符串中解码第一个 Unicode 字符和它的大小：DecodeRuneInString(s string) (r rune, size int)

unicode 包提供的功能：

  • 字符属性和分类：unicode 包提供了一系列的函数和数据表，用于检查 Unicode 字符的属性，如判断是否为数字：IsDigit(r rune) bool
  • 大小写转换：提供了函数用于转换字符的大小写，这些转换考虑到了特定语言的规则。如字符大写转换：ToUpper(r rune) rune
  • 脚本分类：能够根据字符属于 Unicode 中定义的哪个脚本（如拉丁、西里尔、阿拉伯等）来分类字符。

在处理 Unicode 文本和字符编码时，这两个包提供基础且必需的功能。unicode/utf8 包更侧重于与 UTF-8 编码相关的操作，而 unicode 包提供了更广泛的关于字符属性和分类的功能。
