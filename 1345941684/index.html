Go 语言常量

发表于 2024-08-15 更新于 2024-08-16 分类于 Go ， 3.核心概念 阅读次数： 本文字数： 1.7k 阅读时长 ≈ 6 分钟

基本概念

常量（Constant）指在整个程序生命周期内都不变的值，通常是一些字面量。常量在代码编译预处理阶段会直接展开，所有引用常量的地方都直接替换为常量实际值。也就是说，编译后常量已完全消失，因此不能对常量取指针地址，更不可能在运行时修改常量值。

Go 语言中，常量用于储存程序中需要反复使用的固定值，如状态码和配置，使用常量能防止这些值被意外修改。

常量声明

常量使用关键字 const 定义，必须提供值：

const ConstantName ConstantType = Expression

  • ConstantName：常量名称。

  • ConstantType：常量类型，可选。如果省略类型，为无类型常量。

  • Expression：常量值或常量表达式，必须在编译时确定。

下面是简单示例：

package main

import (
	"fmt"
)

// 完整常量声明，指定类型为 float32
const Pi float32 = 3.14159265358979323846

// 无类型常量
const Pi1 = 3.14

func main() {
	fmt.Printf("Pi：%v %T\n", Pi, Pi)
	fmt.Printf("Pi1：%v %T\n", Pi1, Pi1)
	fmt.Println(Pi * Pi1) // Pi1 自动转为 float32 类型后运算
}

常量表达式

常量赋值表达式中只能使用内置函数和常量。常量间操作（包括算术、逻辑以及比较操作）会在编译时进行，结果也是常量：

package main

import (
	"fmt"
	"math"
)

// 自定义函数，不能用于常量初始化表达式
func multiplyByTwo(n int) int { return n * 2 }

func main() {
	// 使用内置常量和内置函数初始化常量
	const base = 2
	const pi = math.Pi              // 使用内置数学常量
	const l = float64(len("hello")) // 使用内置函数
	const r = base * pi * l         // 常量之间的操作，编译时计算
	//const i = multiplyByTwo(5)   // 试图使用自定义函数将导致编译错误

	fmt.Println("结果:", r)
}

常量组

常量组是一种语法结构，允许将多个常量组合在一起声明。常量组使用 const 关键字后跟一对括号来声明，括号内每个常量定义占一行。常量组内的常量如果没有显式赋值，默认采用上一个常量的表达式和类型：

package main

import "fmt"

func main() {
	const (
		a, b, c         = 10, "abc", '啊' // 常量组内能使用并行赋值
		x       float32 = 100            // 支持类型声明
		y               = a * x          // 常量表达式赋值，a 转为 float32 类型再计算
		z                                // 继承上个常量 y 的类型和值
	)
	fmt.Println(a, b, c, x, y, z)
	fmt.Printf("%T %T %T %T %T %T", a, b, c, x, y, z)
}

上面用括号括起来的多行书写方法叫因式分解关键字，同样用于变量和类型声明。

无类型常量

无类型常量（untyped constant）在声明时没有明确类型，直到在特定上下文中使用时才会被隐式转换为具体类型。例如一个无类型常量的字面量是 42，可以赋值给不同类型变量：

package main

import "fmt"

const x = 42 // 无类型常量，默认推断为无类型整数

func main() {
	var a int = x     // x 在这里转为 int
	var b float64 = x // x 在这里转为 float64
	var c byte = x    // x 在这里转为 byte

	fmt.Println(a, b, c)                    // 输出: 42 42 42
	fmt.Printf("%T %T %T %T\n", a, b, c, x) // 输出: int float64 uint8 int
}

无类型常量常用来储存任意精度的数值，只有将它赋值给特定类型变量时，才会检查和限制值范围：

package main

func main() {
	const Huge = 1 << 10   // 无类型整数 1024
	const i = int8(Huge)   // 编译错误：常量溢出，int8 装不下
	const j = uint64(Huge) // 编译通过
}

枚举

枚举指一系列相关的常量。如果一个常量只有几种可能值，则可将其定义为枚举类型。Go 语言没有枚举类型（enum），而是通过常量组实现枚举。

定义枚举

常量组中使用 iota 关键字来模拟枚举行为，iota 从 0 开始计算，按行自动递增。这种定义常量方法能减少重复代码和出错机率，例如对于星期的枚举包含 7 个值：

package main

import "fmt"

func main() {
	const (
		_ = iota // 忽略起始 0
		Sunday
		Monday
		Tuesday
		Wednesday = iota // 不报错，但是非常规用法
		Thursday
		Friday
		Saturday
	)
	fmt.Println(Sunday, Monday, Tuesday, Wednesday, Thursday, Friday, Saturday) // 输出 1 到 7
}

简单常量组中会省略后续 iota 关键字。此外也可自由用在常量表达式中：

package main

import "fmt"

func main() {
	const (
		a              = iota     // 初始值为 0
		b float64      = iota + 1 // 因为 iota 递增到 1，所以等于 1+1
		c                         // 和上面一样，等于 iota + 1 = 2+1
		d = iota * 2.1            // iota 值递增到 3，等于 3*2.1
	)
	fmt.Println(a, b, c, d) // 输出：0 2 3 6.3
}

不同常量组中 iota 独立计数。如果在同一行中多次引用 iota，其值也是相同的，只会按行递增：

package main

import "fmt"

func main() {
	const (
		a = "a"
		b = iota // 尽管前面没有 iota，但 iota 只管计算行数，所以此时值为 1
		_        // 继续递增为 2，但是跳过值
		d = 'd'
		e        // 等于 d，没有 iota 什么事
		f = iota // 显式恢复 iota 自增，值为 5
	)
	fmt.Println(a, b, d, e, f) // 输出：a 1 100 100 5

	// 新常量组，iota 重新计数
	const (
		g, h = iota, iota * 10 // 第一行中，两个 iota 均为 0
		i, j                   // 第二行中，两个 iota 均为 1
	)
	fmt.Println(g, h) // 输出：0 0
	fmt.Println(i, j) // 输出：1 10
}

位掩码

位掩码（Bit Masks）使用整数值的各个二进制位来表示不同状态，使用 iota 结合位移操作 << 可以轻松创建位掩码常量：

  • 唯一性：每次位移操作都会创建一个二进制中只有一个位是 1 的数字。
  • 易操作：由于每个常量都是一个位掩码，可以使用位运算组合和比较实现多个选项组合。
  • 效率高：由于每个标志仅占用一个位，使用位掩码可以有效地使用内存，并且位运算速度比普通运算快。

位掩码常用于定义权限标志，每个枚举值都是前一个的两倍：

package main

import "fmt"

// 自定义权限类型只能读取预定义值
type FilePermissions int

// 组合后能表达数字 1 到 7，代表不同权限
const (
	Read    FilePermissions = 1 << iota // 1 << 0 = 1 二进制 0001
	Write                               // 1 << 1 = 2 二进制 0010
	Execute                             // 1 << 2 = 4 二进制 0100
)

func main() {
	// 设置权限为读和写
	myPermissions := Read | Write
	fmt.Println("Permissions:", myPermissions) // 输出 3

	// 检查是否有运行权限
	fmt.Println("Execute Permission?", myPermissions&Execute != 0) // 返回 false
}

也可以方便地生成储存单位常量：

package main

import "fmt"

type ByteSize float64

const (
	_           = iota
	KB ByteSize = 1 << (10 * iota)
	MB
	GB
)

func main() {
	fmt.Printf("1 KB = %.f Bytes\n", KB) // 输出：1 KB =  1024 Bytes
	fmt.Printf("1 MB = %.f Bytes\n", MB) // 输出：1 MB =  1048576 Bytes
	fmt.Printf("1 GB = %.f Bytes\n", GB) // 输出：1 GB =  1073741824 Bytes
}
